HTML Interview Levels Summary

LEVEL 1 — HTML BASICS (Foundation Stage)

TopicDescriptionExample
HTML StructureUnderstanding <!DOCTYPE html>, <html>, <head>, <body>.<!DOCTYPE html><html><head></head><body>Hello</body></html>
Basic Tags<h1>-<h6>, <p>, <br>, <hr>, <b>, <i>, <u><h1>Heading</h1> <p>Paragraph</p>
ListsOrdered <ol> and unordered <ul> lists with <li> items.<ul><li>Item 1</li></ul>
Links & ImagesAnchor tags <a> and images <img> with alt text.<a href=\"url\">Link</a>, <img src=\"img.jpg\" alt=\"Desc\">
CommentsAdding comments for documentation.<!-- This is a comment -->
Block vs InlineUnderstanding display behavior of elements.<div>Block</div>, <span>Inline</span>

LEVEL 2 — INTERMEDIATE HTML (Practical Stage)

TopicDescriptionExample
HTML Attributesid, class, style, title to control element behavior and styling.<div id=\"main\" class=\"container\" style=\"color:red\" title=\"Info\">Text</div>
TablesStructure tabular data using <table>, <tr>, <td>, <th>, <thead>, <tbody>.<table><tr><td>Data</td></tr></table>
Forms & InputCreating forms with <form>, <input>, <textarea>, <select>, <option>, <label>, <button>. Input types like text, email, password, etc.<form><input type=\"text\" required placeholder=\"Name\"></form>
Form Attributesmethod, action, required, placeholder, pattern, readonly to control form behavior.<form method=\"post\" action=\"submit.php\"></form>
Media TagsEmbedding audio and video using <audio> and <video> tags.<audio controls src=\"audio.mp3\"></audio>
Linking CSS & JSInclude external stylesheets and scripts with <link> and <script>.<link rel=\"stylesheet\" href=\"style.css\"><script src=\"app.js\"></script>

LEVEL 3 — ADVANCED HTML5 (Modern Features)

TopicDescriptionExample
Semantic ElementsDefine meaning of content: <header>, <footer>, <article>, <section>, <nav>, <aside>, <main>.<header>Header</header>
New Form InputsHTML5 input types: email, date, range, color, number, etc.<input type=\"email\" placeholder=\"Enter email\">
MultimediaEmbedding audio/video with controls.<video controls src=\"video.mp4\"></video>
GraphicsDraw using <canvas> or vector graphics using <svg>.<canvas id=\"myCanvas\"></canvas>
StoragelocalStorage (persistent) vs sessionStorage (tab session).localStorage.setItem('key','value');
Geolocation APIGet user location via navigator.geolocation.navigator.geolocation.getCurrentPosition()
Drag and Drop APIEnable drag-and-drop interactions in browser.element.draggable = true;
Responsive DesignUse <meta name=\"viewport\"> for mobile compatibility.<meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0\">
Microdata / SEO TagsMeta tags for SEO & title, description tags.<meta name=\"description\" content=\"Site info\">

LEVEL 4 — EXPERT STAGE (Performance & SEO)

TopicDescriptionExample
SEO TagsUse <title>, <meta name=\"description\">, <meta name=\"keywords\">, proper heading structure.<title>Page Title</title>
AccessibilityUse alt attributes, ARIA roles, semantic headings for accessibility.<img src=\"img.jpg\" alt=\"Description\">
Best PracticesSemantic tags, minimize inline CSS, optimize images, lazy loading, proper nesting.<img src=\"img.jpg\" loading=\"lazy\">
FaviconUse <link rel=\"icon\" href=\"favicon.ico\"> to set favicon.<link rel=\"icon\" href=\"favicon.ico\">
Performance TipsUse CDN, compress media, defer/async scripts, avoid large inline scripts.<script src=\"app.js\" defer></script>
